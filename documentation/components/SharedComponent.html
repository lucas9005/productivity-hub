Description

This component serves as a template for creating other shared components and provides a consistent base structure for shared UI elements.

The component demonstrates:

  • Standalone component configuration for modular architecture
  • Integration with CommonModule for basic Angular directives
  • Basic component structure with Tailwind CSS styling
  • Reusable template structure for consistent UI patterns

Import and use the component in your templates:

Example :
import { SharedComponent } from './shared/shared.component';

@Component({
  standalone: true,
  imports: [SharedComponent],
  template: `<app-shared></app-shared>`
})

To extend this component with custom content:

Example :
@Component({
  selector: 'app-custom',
  standalone: true,
  imports: [SharedComponent],
  template: `
    <app-shared>
      <div class="custom-content">
        <h3>Custom Section</h3>
        <p>Your custom content here</p>
      </div>
    </app-shared>
  `
})
export class CustomComponent {}
